## Description

**OpenXRExtensionWrapperExtension** allows implementing OpenXR
extensions with GDExtension. The extension should be registered with
`OpenXRExtensionWrapper.register_extension_wrapper()<class_OpenXRExtensionWrapper_method_register_extension_wrapper>`{.interpreted-text
role="ref"}.
